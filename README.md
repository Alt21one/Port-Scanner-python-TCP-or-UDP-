# Python Port Scanner (GUI)

Simple multithreaded TCP and UDP port scanner with GUI.

## Features

- TCP scanning
- UDP scanning
- Scan local or network devices

## How to run
```bash
python port_scanner.py
```
## Example 

Protocol:
```TCP / UDP / BOTH```

IP address:
```127.0.0.1 or localhost```
